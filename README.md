### Practical 5 – WordPress Child Theme

In this practical I learned how to create and customise a WordPress child theme. I used my startup website idea called **MindMate AI**, which focuses on providing mental health support for university students.

First, I created a child theme based on the **Twenty Twenty-Five theme**. A child theme allows changes to be made to the design of a website without modifying the original parent theme.

After creating the child theme, I customised several parts of the website. I added a **logo**, edited the **site title**, and created a **navigation menu** with pages such as Home, Features, Team, and Contact.

I also improved the homepage by adding different **content sections and images** that explain the purpose of MindMate AI. These sections describe features such as anonymous AI support, guided breathing exercises, mood tracking, and links to university counselling services.

Through this task I learned how WordPress themes work, how child themes inherit styles from the parent theme, and how the block editor can be used to customise the layout of a website.

Overall, this practical helped me understand how to design and customise a professional-looking website using WordPress.
