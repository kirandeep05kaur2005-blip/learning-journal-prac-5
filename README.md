This week I learned how to create and customise a WordPress child theme as part of my web development studies.

I built a small startup website called **MindMate AI**, which focuses on providing mental health support for university students. Through this project I learned how child themes work in WordPress and how they allow developers to customise a website while keeping the parent theme unchanged.

It was interesting to see how design elements such as navigation menus, images, and layout sections can be customised using the WordPress block editor.

This practical helped me improve my understanding of website structure and theme customisation.
